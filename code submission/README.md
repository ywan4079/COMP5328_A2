Instruction of running code
1. Run `python -m venv venv` to create a virtual environment, or other methods you prefer.
2. Activate the virtual environment
3. Run `pip install -r requirements.txt` to install the packages
4. Open model training.ipynb and run all of the cells to train the model and see their top-1 accuracy.
5. Run `python visualize results.py` for more detailed plots.
